# Производственная практика АО «Куйбышевский нефтеперерабатывающий завод»
Данный репозиторий содержит программный модуль для автоматизации расчета выполнения сменного задания на нефтеперерабатывающем предприятии АО «Куйбышевский НПЗ».

## Функционал программы
- Запрос у пользователя данных о плановом и фактическом выпуске продукции
- Расчет процента выполнения плана сменного задания
- Определение статуса выполнения плана
- Автоматическая запись результатов смены в файл отчетности

## Стек технологий
- Python 3.9+
